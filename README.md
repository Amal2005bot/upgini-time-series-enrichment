# Sales Forecasting (Time Series)

Time-series sales prediction using feature enrichment and gradient boosting.

## Stack
- Python
- Pandas
- Upgini
- CatBoost

## Pipeline
- Data preprocessing
- Time-based feature enrichment
- CatBoost regression
- SMAPE / MAPE evaluation

## Files
- `.ipynb` – Notebook workflow
- `.py` – Script version
- `train.csv` – Dataset

## Run
pip install pandas upgini catboost
